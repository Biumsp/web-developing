There are block-level elements and inline elements. But we can change their behavior using the "display" property. Inline and block elements have different behavior in terms of box-model properties. Inline elements cannot force vertical properties, like margin and patting in top and bottom sides. On the other hand, block elements can enforce all properties they like on the surrounding elements.

We have another option: inline-block. This is going to behave like an inline element, except, width, height, padding, margin are going to be respected.

Display can also be set to none, s.t. you can hide and unhide elements
