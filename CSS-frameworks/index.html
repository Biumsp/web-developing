CSS Frameworks

CSS frameworks help us quikly build responsive websites. Bootstrap is the most used one.

Bootstrap

Bootstrap consists of pre-bult components and a grid system

To include Bootstrap we can download its stylesheets and js files (and link them directly as we do for our css files), or we can use a cdn (content distribution network) to get them from an hosted version. It's still a link element but the href is a url. We can copy it from here and link it in the head, before our styles. We also need to include the js files at the bottom, before the body closing tag

For a real production application you need to download it

Most of what Bootstrap does is used through classes.

Bootstrap components

  • .container this is the most basic layout element, requird by the grid system. They contain and pad the content
