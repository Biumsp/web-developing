CSS Intro

We style html files in an external css file. To link these two filese we need the link element. This element must be placed in the head. It has an attribute "rel" where you have to put "stylesheet". Finally, it has an attribute href, wher I put the path to the css file. Typically all css files are placed in a "styles" folder.

The core structure of a css file is this:
selector {
property: value;
}

Properties

We can modify many properties, for example:

  • color: it changes the text color

Colors

There are many color systems. Named colors are approx 140. We have rgb system, hexadecimal system (same stuff, but a single number in hexadecimal). We can look for a nice color palette online.
