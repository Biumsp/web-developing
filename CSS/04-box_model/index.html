We can change the width and height of the inner content area: the content box.

Border and border-related properties

The border is the border around an element. The main properties are:
  • border-width
  • border-color
  • border-style
They HAVE to be ALL set. Otherwise you don't see anything.

This is the second one: you can style individually the sides of the border.

This is the third one. You can use one single property ("border") to select the width, style and color all at once.

We usually give border width with px because they are very thin and don't change with the page zoom. The border width is added to the content area's. To change that we have to set
box-sizing: border-box; Doing so, will make width select the overall width: content+border.

Padding

The space between the content and the border. Like border, we have the shorthand "padding" for all the sides at once

Margin

It's the space outsde of the border. The body element has some default margin, just as all h1s. It is pretty common to set body {margin: 0;}, to avoid this default margin.
