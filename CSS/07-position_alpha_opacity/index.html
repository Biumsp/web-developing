Opacity

It's a property that goes from 0 to one (like the alpha value in rgba) and applies to the entire element, also text. Also buttons. Anything inside the element.

Opacity is set to 0.9
Opacity is set to 0.6
Opacity is set to 0.3

Position

It establishes how four other properties behave: top, right, bottom, left

position: static; This is the default value. If an element has this position value, top, right, bottom and left don't do anything.

position: relative; With this value, top, b... modify the position of the element, relatively to where it would be normally

position: absolute; When set like this, the element is removed from the normal display flow, and no space is created for it. It doesn't take up any space. Now the offsets (top, r...) are relative to its closest positioned (so with position != static) ancestor, if any. Otherwise is positioned relative to the initial containing element (body)

position: fixed; Its position becomes relative to the initial containing block. This is the way you block text on the top of the screen.

position: sticky; It is fixed only when the top gets to it
