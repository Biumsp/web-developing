Responsive Design

It is about making a website responsive to different screen sizes or different features of screen and devices.

We do it using media queries: they allow us to modify our style depending on some parameters, like screen width, or portrait/landscape-mode. It is not just about scaling things, but also about changing the layout.

Using the dev-tools of chrome you can also view the page as if it was displayed at a different screen-size They are very well described here

Media queries begin with "@media", folloewd by a criterion "(parameter: value)" like (min-width: 400px), or (orientation: landscape)
