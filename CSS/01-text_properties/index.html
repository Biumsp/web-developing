Text Properties

We have a lot of text properties. We have:

  • text-align: sets how text is horizontally alligned in the content box of the element (blue box)
  • font-weight: this has keywords (normal, bold..) or numbers (400, 900...). Not all fonts have all fonts weights.

Font Family

Fonts are a complex subject. You cannot give for granted that they will be supported across all devices. You can look here for a complete collection of web safe CSS font stacks. But, as you notice, not all fonts are supported on every machine. When you specify a font, you can supply a font-stack: a list of fonts, that you would like to be used, from the best to the worst case scenario. You just specify them as a list after "font-family". For example you can tell, as a last resource, to use any sans-serif font (this is a family).
