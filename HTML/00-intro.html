What is HTML?

Introduction

HTML is a "markup language". It provides informations about the text, s.t. "this is a button" or "this should be bold"

HTML Elements

We perform this markup using markup elements Using those we can make things bold, or italicized, or underlined... More in general we can wrap text in tags, that give them formatting, or a class... in general, they will attach some meta-informations to the text

Some Elements

Paragraphs split out content in sections, that start in different lines

Headings (h1 to h6) are usually ranked by importance, to define headings, sub-headings... They should not be used for the size, but just for conceptual purpose, to give structure. We should always use a single h1 at most. Headings take up a whole line.

HTML document structure

Every HTML document must have a fixed structure. We have to set the DOCTYPE (is an only-open tag).

Then, we have to create the html element, which contains the whole document.

Inside the html element we have the head: it contains the metadata about the document (css files' paths, fonts...).

The title element: it changes the title on the card displayed in the browsers, and it is what search engines look for.

Then, we have a body element: in contains all the content (all of the buttons, all the navigation bars, all the sections...)
