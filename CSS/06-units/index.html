Relative Units

  • em: different behaviours
  • rem: root ems. They derive from the root elemen. The root element is the html element at the beginning

Em

ems are relative units: a font-size of 2em is twice the font-size of the parent element. For padding or margin it is relative to the font-size of the element itself, whatever it was computed to be, using the parent's font-size. For example, if I make a button and I want it to be rounded and have padding, I want this properties to scale with the size.
