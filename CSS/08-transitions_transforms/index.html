Transitions and Transforms

Transitions

We can introduce transitions between changes in styles. You can specify the duration of the transition, its timing function, its delay and the specific properties that will undergo that transitions in that way. Even if I want everything to perform that transition, it is better to single-out the specific properties.

Transforms

We can:

  • transform: rotate(45deg); Rotate of some degrees, or other units.
  • transform: scale(0.8); Scale to 80% along x and y
  • transform: translate(100px, -50px); translate in x and y directions
  • transform: skew(20deg); skew the element
We can change the transform origin. In rotations it changes the pin point. The transform is applied to anything inside.
