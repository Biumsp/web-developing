Tables

A 2D way to display some content. It doesn't need to be text. The teachers' cards in superprof are displayed in a table-linke way

The elements are "td" for a cell, "tr" for table rows, "th" for the headers. We can also divide the header's rows from the body's and the footer's. We use thead, tbody and tfoot.

An example of a table

Animal Average Mass [kg] Flighted
Ostrich 104 No
Somali Ostrich 90 No

We can have headers multiple columns large. You set them by adding the attributes rowspan and colspan.
